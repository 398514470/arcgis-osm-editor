ArcGIS Editor for OpenStreetMap Version 2.1, Server Component

Getting Started

You've installed the ArcGIS Editor for OSM Server Component. What to do next:

  • If your system is Windows Server 2003 or Windows XP and you've installed the Server Component, there is a post-installation step you must do to set the Application Configuration to the aspnet_iisapi.dll. See documentation at Server Component Post-Installation Steps. Other supported operating systems should not require these steps.
  • Read the important notes about using the Feature Service capability at ArcGIS Editor for OSM Feature Service webhelp. These will inform you of how to use the feature services within the OSM community and how to use the editing tools.
  • Launch the interface for creating a feature service based on OpenStreetMap data. The URL for this is http://servername/OSM (replacing servername with the server upon which you installed the server component).
  • See What's New to find out about the new features of this release.

Please use the Discussion Forum to get help. If you want to report a bug or enhancement request, use the Issue Tracker.

For known issues and notes about system environment with this version, see Code In-Progress Notes.
